# 商店分析系统

基于快递助手导出数据进行商店经营分析的统计系统，主要用于拼多多店铺的数据分析和成本核算。

## 📋 项目简介

拼多多自带的推广花费计算并不准确，本项目通过实际发货数据来计算利润与成本，提供更准确的数据分析。支持订单管理、推广花费统计、月度汇总等功能。

## 🛠️ 技术栈

### 后端
- **框架**: Django
- **数据库**: SQLite
- **数据视图**: django_db_views
- **其他**: openpyxl (Excel处理)

### 前端
- **框架**: Vue 3
- **UI组件**: Element Plus
- **图表**: ECharts
- **HTTP**: Axios

## ✨ 功能特性

### 📊 数据分析
- **订单分析**: 按店铺和月份统计收入数据
- **推广花费**: 拼多多推广花费月度统计
- **趋势图表**: 可视化展示收入和花费趋势

### 📈 数据管理
- **数据导入**: 支持Excel文件导入订单和推广数据
- **异步处理**: 后台线程处理大批量数据导入
- **数据汇总**: 自动生成月度汇总视图

### 🎨 界面特性
- **日期筛选**: 支持月份范围选择
- **多店铺**: 支持多店铺数据对比
- **响应式设计**: 适配不同屏幕尺寸

## 🚀 快速开始

### 后端启动

```bash
# 进入后端目录
cd backend/main

# 安装依赖
pip install -r ../requirements.txt

# 运行迁移
python manage.py migrate

# 启动服务
python manage.py runserver
```

### 前端启动

```bash
# 进入前端目录
cd frontend

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

## 📖 使用说明

### 数据导入
1. 在系统中选择要导入的数据类型（订单/推广）
2. 上传Excel文件
3. 系统自动异步处理并导入数据

### 数据查看
1. 访问统计页面
2. 选择日期范围
3. 查看收入和推广花费趋势图表

## 📝 注意事项

- 本项目为学习和练习项目，仅供参考
- 确保导入的Excel数据格式正确
- 大数据量导入可能需要较长时间
